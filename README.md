# Breast-Cancer

## Introduction
This repository solved the problem of classifying mammograms as malignant or normal/benign using deep learning.

## Architechture
 - ConvNeXt
 - CNNs

## Dataset
 - INbreast: 
 - CMMD
 - DDSM
 - Mias
 - RSNA
